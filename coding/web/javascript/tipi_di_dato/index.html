Tipi di dato

I tipi di dato previsti da Ecmascript sono String (le frasi), Number, Boolean (valori vero/falso), Array (un elenco di valori), Object, null e undefined.

Quando si dichiara una variabile e si assegna un valore, allora alla variabile viene associato un tipo di dato. Se si assegna un valore differente, anche il tipo di dato cambia di conseguenza.

Dichiarazione variabili

Dalla versione di Ecmascript 6 le variabili si dichiarano utilizzando la parola chiave let, anche se la parola chiave var usata in Ecmascript 5 è ancora valida. Un esempio di dichiarazione di variabili è la seguente:

  • let naturalNumber = 10;
  • let realNumber = 10.5;
  • let toBeOrNotToBe = true;
  • let word = "paragraph";
  • let vector = ["one", "two", "three"];
  • let thing = null;
  • var anotherThing = null;

In questo esempio abbiamo dichiarato delle variabili che contengono rispettivamente un numero intero, un numero decimale, un valore booleano, una stringa, un vettore e due oggetti. In particolare, i due oggetti hanno un valore iniziale null che sta ad indicare un oggetto non ancora in memoria.

Tutte queste variabili potevano essere dichiarate anche con la parola chiave var, come avvenuto per l'ultima variabile dichiarata nell'esempio precedente.

Tra i casi particolari, troviamo il valore undefined, utilizzato per indicare una variabile che non è stata dichiarata.

Un altro caso particolare è il valore Nan, utilizzato per indicare un risultato che Non è un Numero (dall'inglese Not a Number). Questo valore è assegnato quando si tentano operazioni del tipo "radice quadrata di meno uno", che non ha risultato nell'insieme dei numeri reali.

Altri casi particolari sono i valori POSITIVE_INFINITY e NEGATIVE_INFINITY, utilizzatio per indicare risultati troppo grandi per poter essere rappresentati da una variabile numerica.

E' possibile creare anche una versione ad oggetti (detta "wrapper") delle variabili indicate:

  • var naturalNumberObj = new Number(10);
  • let realNumberObj = new Number(10.5);
  • let toBeOrNotToBeObj = new Boolean(true);
  • let wordObj = new String("paragraph");
  • var vectorObj = new Array("one", "two", "three");

Come si nota, gli oggetti sono creati con la parola chiave new che serve ad inserire un oggetto in memoria. L'operazione di inserimento di un oggetto in memoria è detta allocazione, quindi tecnicamente un oggetto è allocato in memoria. Come già detto, alla variabile si assegna un valore null se non si vuole allocare l'oggetto in memoria.

Identificazione del tipo di dato di una variabile

E' possibile identificare il tipo di una variabile utilizzando la parola chiave typeof. Partendo dalle variabili dichiarate in precedenza, si possono fare i seguenti esempi:

  • typeof naturalNumber; restituisce number
  • typeof realNumber;restituisce number
  • typeof toBeOrNotToBe;restituisce boolean
  • typeof word;restituisce string
  • typeof vector;restituisce object
  • typeof thing;restituisce object
  • typeof anotherThing;restituisce object
  • typeof variabileNonDichiarata;restituisce undefined

Come si può notare, sia la variabile oggetto, sia la variabile dichiarata come array, sia tutti i tipi "wrapper" sono di tipo oggetto.

Una precisazione però è da fare sulla variabile inizializzata a null: la specifica di Ecmascript prevede che, nonostante la variabile sia da considerare come tipo "null", l'operazione typeof null restituisca object.

Un'altro risultato non intuitivo è il tipo risultante da una variabile non dichiarata: prendendo ad esempio la variabile y (non dichiarata), la valutazione del tipo di questa variabile, tramite l'operazione typeof y, restituisce il valore undefined che è appunto utilizzato per indicare variabili non dichiarate.

E' necessario a questo punto una definizione di "oggetto" in ambito informatico: un oggetto è un'entità che contiene proprietà e metodi. Un'esempio di oggetto può essere una persona, che contiene proprietà quali nome, cognome, data di nascita, ecc..

    
        let persona = {
            firstname: "John",
            lastname: "Doe",
            id: 5566
        };

Nei prossimi capitoli si vedrà come utilizzare gli oggetti, per ora è importante avere un'idea di cosa siano.

Conversioni tra tipi

Per convertire dati dal formato stringa al formato intero, si usa l'istruzione parseInt(), come nell'esempio seguente:

    
        parseInt("123.45")   // restituisce 123
        parseInt("123.45KG") // restituisce 123
        parseInt("aaaa")     // restituisce NaN
        parseInt("0.1e6")    // restituisce 0 perchè legge solo il numero intero 0
        parseInt("0x77")     // restituisce 119 (= 7*16 + 7)

E' possibile convertire la stringa in un numero con una base specifica, come nell'esempio seguente:

    
        parseInt("ZR",36) // restituisce 1289 (= 35*36 + 29)

Per convertire dati dal formato stringa al formato float, si usa l'istruzione parseFloat(), come nell'esempio seguente:

    
        parseFloat(".3")    // restituisce 0.3
        parseFloat("0.1e6") // restituisce 100000

Per i numeri decimali, non è prevista la conversione del numero in una base specifica.
