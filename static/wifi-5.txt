Others icons created by Inkscape:

To resize the SVG image:
  Select the image, resize with CTRL+mouse (CTRL key let you keep the aspect ratio);
To set the document width, height and margins:
  Select File -> Document Property, click on Resize Page to fit contents, set width, height and margins

To export in other formats, use Inkscape and ImageMagik script

flatpak run org.inkscape.Inkscape  wifi-5.svg --export-width=32 --export-filename="favicon.png"
convert favicon.png favicon.ico

flatpak run org.inkscape.Inkscape  wifi-5.svg --export-width=512 --export-filename="icon-512.png"
flatpak run org.inkscape.Inkscape  wifi-5.svg --export-width=192 --export-filename="icon-192.png"
flatpak run org.inkscape.Inkscape  wifi-5-apple.svg --export-width=180 --export-filename="apple-touch-icon.png"
